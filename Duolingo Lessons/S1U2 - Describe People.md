Japanese doesn't have words like a or the.
For example, the word *sen-se-i* can mean both teacher and a teacher.

| Japanese         | English Meaning |
| ---------------- | --------------- |
| se-n-se-i        | teacher         |
| be-n-go-shi      | lawyer          |
| i-sha            | doctor          |
| be-n-go-shi desu | She's a lawyer  |
| ka-kko-i-i       | cool            |
| ya-sa-shi-i      | nice            |
| hi-to            | person          |
| ga-ku-se-i       | student         |
